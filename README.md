# Aplikasi Budget Challenge 30 Hari

Aplikasi sederhana untuk membantu mengatur pengeluaran dengan tantangan budget 30 hari.

## Fitur
- Input pengeluaran harian
- Tentukan budget harian
- Lihat sisa budget
- Grafik pengeluaran

## Cara Menjalankan
1. Install library:
   pip install -r requirements.txt
2. Jalankan:
   streamlit run app.py
